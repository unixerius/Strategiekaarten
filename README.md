# Strategiekaarten

Een plan van aanpak, of "strategiekaarten", voor deelnemers die een steuntje in de rug kunnen gebruiken bij hun deelname aan Unixerius-cursussen.

Elke PDF in deze repository is een document dat dubbelzijdig kan worden geprint. De voorkant geeft een stappenplan voor één van de taken uit de Unixerius-cursussen. De achterzijde bevat, waar mogelijk, een voorbeeld van hoe het stappenplan zou worden uitgewerkt.


# Auteursrecht, dankbetuiging en meer

De term "strategiekaart" is afkomstig uit het werk van [het _Geef me de 5_ team](https://www.geefmede5.nl). Zij hebben een boek gemaakt, waarin leerlingen stappenplannen ("strategiekaarten") vinden voor vaak-voorkomende schooltaken. Dit boek is [te koop op hun website](https://www.strategiekaartenhuiswerk.nl).

Tess is door dit boek geïnspireerd om PvA-kaarten ("plan van aanpak") te maken voor een aantal taken die haar leerlingen in de les krijgen. Voordat deze kaarten vrijelijk beschikbaar werden gesteld is er overlegd met Colette, Elisa en het Gmd5-team. Unixerius is dankbaar voor hun toestemming om deze kaarten vrij te delen. 

De PvA-kaarten in deze Github repository zijn beschikbaar gesteld onder de _Creative Commons_ CC-BY-SA licentie. [Lees hier alle details.](https://github.com/Unixerius/Strategiekaarten/LICENSE.txt) Kortgezegd: deze kaarten mag je gratis gebruiken en verspreiden, maar **niet** voor commerciële doeleinden gebruiken.
